Report

The report form will show the "accurate" stat points pr. level for the active enemy.

Controls:

  • Grid
  • Tools

Grid

 

In the grid you can the levels are displayed vertically downwards, (Y axis) and the stats are shown horizontally (X axis). The form will be updated if you alter the stats in the main form, or if the form is open and you open a new file. If you open a new file while this form is open, the new statics will be loaded into the form. You can not alter the values inside the grid. Tools

 

At the top of the form you can alter the display with various tools. Average party level and Finished char. level only affect experience points. Finished char. represents the character that caused the final blow to the enemy and affects extra exp. Extra exp, as you may know, are only given to this character.

 

EXP formulas:

EXP:
exp_value * (5 * (x - average_party_level) / average_party_level + 4)

Extra EXP:
exp_extra_value * (5 * (x - finish_char_level) / finish_char_level + 4)
