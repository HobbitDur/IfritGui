Ifrit

2009 © Omega corporation

Ifrit is used to edit enemies for Final Fantasy VIII PC version. You need to extract the files from battle.fs  (located in: ...\FINAL FANTASY VIII\Data\). It can not edit the battle.fs directly (yet). You need to open c0mXXX.dat files (from c0m001.dat to c0m143.dat. NOT c0m127.dat).

IFRIT CAN: 

Edit enemy names.

  1. Alter starting level of medium and high levels.
  2. Change elemental and status properties.
  3. Edit general stats.
  4. Drop and mug items.
  5. Drawn magic.

 

IFRIT CAN NOT:

  1. Edit scan text.
  2. Edit AI.
  3. Change enemy formations.
  4. Alter attacks and moves.
