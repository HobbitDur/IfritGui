Specs

Name & Levels

The enemy's name can be altered here. Please note that not all values are known (to me). Take caution when altering names. Ifrit will attempt to show the real name if you type in unknown values, displayed to the right of the name text box.

Concerning the starting level of the enemy's medium and high levels: These levels will affect what the enemy will drop at certain levels, and what to steal and draw. 

Elements and statuses

The enemy can absorb, resist, nullify or be weak against one or more elements. When dragging the arrow in the track bar you will see how the percentage changes. The elemental rate can only be dividable by 10.

 

100% : Normal damage

10% - 90% : Percentage of damage. Damage is in other words reduced for this element.

0% : The enemy is immune to this element.

-X% (negative value) :  The enemy will absorb X% of the damage, in other words heal it.

100%+ : The enemy becomes weak towards this element.

 

Under the elements you can specify each status resistance in %.

Stats

In FFVIII the enemies pretty much follows your level. Thus their stats can change too. Square decided to base 4 variables on each stat to get the actual value at each level. The actual values follow these formulas (x is a value of monster level):

 

HP:
(hp_1 * x * x / 20) + (hp_1 + hp_3 * 100) * x + hp_2 * 10 + hp_4 * 1000

STR:
(x * str_1 / 10 + x / str_2 - x * x / 2 / str_4 + str_3) / 4   *PLEASE NOTE: I'm not 100% sure on the STR formula, but it should be accurate enough to get the general idea.

MAG:
exactly like STR

VIT:
x / vit_2 - x / vit_4 + x * vit_1 + vit_3

SPR, SPD, EVA:
Exactly like VIT

EXP:
exp_value * (5 * (x - average_party_level) / average_party_level + 4)

Extra EXP:
exp_extra_value * (5 * (x - finish_char_level) / finish_char_level + 4)

Show stats pr. level

Clicking this button will open a new dialog (form) displaying the actual stat values for each level. You can choose how many levels you want to view at a time. Changing values in the main form will update this form without closing it. Note that only exp. and extra exp. are the only values affected by the party's level.

Items

 

Mugrate and droprate are how likely it is in % the drop or mug will occur.

Each enemy holds up to 4 different items in drop and mug. However only 1 of these items may be claimed. To the right of each item the quantity is displayed and may be changed as well. The different drops also depends on Low, Medium and High levels.

 

Draw

Each enemy can hold max 4 magics to draw. Depending on which level the enemy are at, the enemy may hold different magics. Like items there are Low, Medium and High levels. The '??' are unknown values (for me), and editing these will not change when you save the file. I merely made this to see if something other than 0 ever were loaded.
